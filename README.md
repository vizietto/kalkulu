# kalkulu
Kalkulu is an interpreter for the Wolfram language, written in literate Haskell.

# Quickstart
To run the REPL, simply use `cabal run`. To render the source code
as a PDF file, do
```
cd src/
pdflatex -shell-escape main.tex
```
